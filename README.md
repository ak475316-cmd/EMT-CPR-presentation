# EMT 教材

這個專案用來整理 EMT 教材、課程素材、教案草稿與後續輸出檔。

## 目錄

- `docs/`：課程規劃、教材筆記、製作紀錄
- `src/`：可重複產生教材的來源檔或腳本
- `tests/`：檢查教材產出或腳本行為的測試

## 目前狀態

專案已先建立中性骨架，尚未選定特定技術棧。後續可以依需求加入：

- PowerPoint / 文件產出流程
- HTML 互動教材
- 題庫、案例情境或技能站素材
- 自動化檢查與發布流程

## 開始工作

1. 把原始教材或規劃筆記放到 `docs/`。
2. 如果有可重複產生的程式或模板，放到 `src/`。
3. 如果建立自動化流程，再補上對應的測試到 `tests/`。
